## 📖 About This Project

WolfPort is my personal portfolio website showcasing my skills, projects, and experience as a Full-Stack Developer. Built with React and hosted on Vercel, it features an AI-powered chatbot using Google's Gemini API that can answer questions about my work, web development, and general knowledge.

### 🎯 Purpose

- Showcase my professional work and projects
- Demonstrate technical skills in modern web development
- Provide an interactive experience with AI chatbot
- Serve as a central hub for professional contact
